# AzurLaneAutoScript 介绍

::: tip 💡 什么是 Alas？
**AzurLaneAutoScript（简称 Alas）** 是一款专为《碧蓝航线》设计的全自动挂机脚本，旨在帮助玩家自动化处理游戏中的重复性操作，让你从繁琐的日常任务中解放出来。
:::

## ✨ 核心特性

<div class="vp-features">

### 🎮 多服多设备支持
- 支持国服、日服、国际服、台服等多个服务器
- 兼容各类安卓模拟器（蓝叠、夜神、雷电等）
- 支持云手机、实体手机等多种部署方式
- 自适应不同分辨率和屏幕比例

### 📦 功能覆盖全面
- **主线图**: 自动推图、三星通关、捞船
- **活动图**: 自动刷活动 PT、捞活动船
- **每日任务**: 日常、周常、演习、每日副本
- **大型作战**: 自动开荒、清体力、买猫箱
- **后宅与学院**: 自动喂食、学习技能
- **科研与建造**: 自动科研、建造、拆解

### 🚀 稳定可靠
- 专为 7×24 小时长时间运行设计
- 智能异常检测与自动恢复
- 低资源占用，不影响正常使用电脑
- 完善的日志记录与错误报告

### 💬 社区支持
- 活跃的开源社区，持续更新维护
- 详细的文档与教程
- 多渠道技术支持（GitHub Issues、QQ群等）
- 定期更新适配游戏新版本

</div>

## 🎯 设计理念

::: warning 减少游戏时间，而非增加
Alas 的设计初衷是**减少玩家花费在碧蓝航线上的时间**，而不是让玩家更"肝"。我们相信：

> "把重复性操作交给 Alas，把游戏乐趣留给自己"
:::

## 🛠️ 技术架构

::: details 点击查看技术细节

Alas 采用 Python 开发，核心架构包括：

- **图像识别**: 基于 OpenCV 的模板匹配与 OCR
- **设备控制**: ADB 与 MuMu 等多种控制协议
- **任务调度**: 智能优先级队列与资源管理
- **配置系统**: 可视化界面，易于上手

:::

## 🌟 为什么选择 Alas？

1. **开源免费**: 完全开源，无需付费，无广告
2. **安全可靠**: 本地运行，不修改游戏数据
3. **持续更新**: 紧跟游戏版本，快速适配新内容
4. **易于使用**: 图形化界面，无需编程基础
